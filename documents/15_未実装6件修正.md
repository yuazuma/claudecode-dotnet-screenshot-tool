## F-05-11: ファイル名トークン修正

**修正ファイル**: `src/AutoScreenshot/Services/FileStorage.cs`

`BuildFileName()` で `evt.Type.ToString().ToLowerInvariant()` をそのまま使っていたため、要件と異なるトークン名になっていた。`TriggerToken()` メソッドを追加してマッピングを実装。

```csharp
private static string TriggerToken(TriggerType type) => type switch
{
    TriggerType.ManualCapture      => "manual",
    TriggerType.MouseLeftClick     => "click",
    TriggerType.MouseRightClick    => "rightclick",
    TriggerType.MouseMiddleClick   => "middleclick",
    TriggerType.MouseDragDrop      => "drag",
    TriggerType.MouseWheel         => "scroll",
    TriggerType.Keyboard           => "keyboard",
    TriggerType.ActiveWindowChange => "windowchange",
    TriggerType.ScreenDiff         => "diff",
    _                              => type.ToString().ToLowerInvariant(),
};
```

## F-05-21: ディスク低容量→自動一時停止

**修正ファイル**: `src/AutoScreenshot/Services/FileStorage.cs`, `NotifyIconWrapper.cs`

`FileStorage` に `Action? OnLowDiskSpaceDetected` プロパティを追加。`CheckDiskSpace()` でしきい値を下回ったときにコールバックを発火する。

`NotifyIconWrapper.Initialize()` で `_storage.OnLowDiskSpaceDetected = OnLowDiskSpace;` を設定。`OnLowDiskSpace()` メソッドでは `Dispatcher.BeginInvoke` 経由で `_paused = true` / `_orchestrator.SetPaused(true)` / トレイアイコン変更を実行。
