## 修正内容

### 1. Notifier スレッドセーフ化

**問題**: `OnCaptured()` が `Task.Run` バックグラウンドスレッドから呼ばれるにもかかわらず、`NotifyIcon` (WinForms UI) を直接操作していた。

**修正ファイル**: `src/AutoScreenshot/Services/Notifier.cs`

- カウンター（`_todayCount`, `_countDate`）を `lock (_countLock)` で保護
- `NotifyIcon` 操作を `System.Windows.Application.Current.Dispatcher.BeginInvoke()` でUIスレッドにマーシャリング
- `FlashIcon()` 内のタイマーコールバックも同様に `Dispatcher.BeginInvoke` でラップ
- `SetPausedState()` / `ShowDiskWarning()` も Dispatcher 経由に統一

### 2. OnDiffTimer エラー抑制

**問題**: RDP・仮想環境でディスプレイハンドルが無効な場合、`CopyFromScreen` が `Win32Exception (NativeErrorCode == 6)` をスローし、3秒ごとに `[ERR]` ログが大量出力されていた。

**修正ファイル**: `src/AutoScreenshot/Services/DiffDetector.cs`

```csharp
catch (Win32Exception ex) when (ex.NativeErrorCode == 6)
{
    Log.Debug("差分検知スキップ (ディスプレイ利用不能): モニタ{Idx}", i);
    return changed;
}
```

`using System.ComponentModel;` を追加して `Win32Exception` を使用可能にした。

### 3. クールダウン UI 追加

**問題**: 設定画面にクールダウン秒数を調整する UI がなく、config.json を直接編集する必要があった。

**修正ファイル**: `src/AutoScreenshot/Views/SettingsWindow.xaml`, `SettingsWindow.xaml.cs`

- 撮影トリガータブにクールダウンセクション追加
- 6本のスライダー: `SldrCooldownClick`, `SldrCooldownDrag`, `SldrCooldownWheel`, `SldrCooldownKeyboard`, `SldrCooldownWindow`, `SldrCooldownDiff`
- `LoadSettings()` / `ApplySettings()` 双方にバインド追加

### 4. DragThresholdMs / WheelIdleMs の config.json 保存

**問題**: `System.Text.Json` は既存 JSON に含まれないフィールドを `Deserialize` 時にデフォルト値で埋めるが、上書き保存しない。このため新規追加フィールドが config.json に永続化されなかった。

**修正ファイル**: `src/AutoScreenshot/Services/ConfigStore.cs`

```csharp
_config = JsonSerializer.Deserialize<AppConfig>(json, _jsonOptions) ?? new AppConfig();
Log.Information("設定を読み込みました: {Path}", path);
Save();  // 追加: 新規フィールドを config.json に書き出す
```

### 5. ImageOverlay 削除（未実装機能の除去）

**問題**: `MetadataConfig` に `ImageOverlay` プロパティが残っていたが、実装されておらず config.json にも出力されていた。

**修正**: `AppConfig.cs` の `MetadataConfig` から `ImageOverlay` プロパティを削除。

### 6. ディスク容量警告トースト通知

**問題**: `FileStorage.CheckDiskSpace()` は警告ログを出力するのみで、ユーザーへの通知手段がなかった。

**修正ファイル**: `src/AutoScreenshot/Services/FileStorage.cs`, `Notifier.cs`, `NotifyIconWrapper.cs`

- `FileStorage` に `Notifier?` フィールドと `SetNotifier()` メソッドを追加
- `CheckDiskSpace()` で `_notifier.ShowDiskWarning(freeMb)` を呼び出し（10分間隔制限付き）
- `Notifier.ShowDiskWarning()` は `NotifyIcon.ShowBalloonTip()` でトースト表示
- `NotifyIconWrapper.Initialize()` で `_storage.SetNotifier(_notifier)` を追加

### 7. build.bat 刷新

**問題**: `build_phase3.bat` という名前で Phase 3 固有のスクリプトが残っており、publish の手順が不明確だった。

**修正**: `build.bat` を新規作成。引数なしでリリースビルド、`publish` 引数で単一 exe をパブリッシュ。

ビルド成功（警告: ImageSharp 3.1.7 既知脆弱性のみ、エラー0件）。
