# 追加要件定義：操作手順説明動画の自動生成

## 日時
2026-05-27

## 作業内容

ヒアリングにより追加機能「操作手順説明動画の自動生成」の要件を収集し、
`requirements/追加要件_手順動画作成支援.md` に要件定義書を書き出した。

## ヒアリング結果サマリー

| 項目 | 決定事項 |
|---|---|
| 出力フォーマット | APNG + MP4（アニメーション GIF は対象外） |
| 動画の構成単位 | セッション単位 / チャプター単位を設定で選択 |
| フレーム表示時間 | 固定秒数 / 実時間再現 を設定で選択 |
| 音声 | Windows SAPI（SpeechSynthesizer）による TTS ナレーション |
| 強調表示 | クリック位置に波紋アニメーション ＋ 操作対象コントロールに矩形枠 |
| テロップ内容 | イベント種別・入力テキスト・ステップ説明文・タイムスタンプ（各項目 ON/OFF 可） |
| 出力解像度 | 設定で選択（原寸 / FHD / HD） |
| 出力フォルダ | 専用フォルダを設定で指定（既定: %USERPROFILE%\Videos\AutoScreenshot） |
| MP4 エンコード | MediaFoundation（OS 標準 API）、追加バイナリなし |
| 生成タイミング | 手順書生成と同時 / 手動ボタンを設定で切り替え（既定: 手動） |
