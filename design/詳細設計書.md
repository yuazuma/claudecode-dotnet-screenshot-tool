## 2. 起動・終了フロー

### 起動シーケンス（App.OnStartup）

```
1. Mutex("AutoScreenshot_SingleInstance", true) → createdNew チェック
   → false の場合: MessageBox 表示後 Shutdown() して終了
2. ShutdownMode = OnExplicitShutdown（WM_CLOSE でのアプリ終了を抑止）
3. ConfigureLogging():
   - ログディレクトリ: %APPDATA%\AutoScreenshot\logs\
   - ファイルパターン: app-.log（RollingInterval.Day、30 日保持）
   - 出力テンプレート: "{Timestamp:yyyy-MM-dd HH:mm:ss.fff} [{Level:u3}] {Message:lj}{NewLine}{Exception}"
4. new NotifyIconWrapper()（依存グラフ構築）
5. NotifyIconWrapper.Initialize():
   a. アイコン生成（IconFactory）
   b. NotifyIcon 作成・表示
   c. HookService.Start()（マウス・キーボード・WinEvent フック設置）
   d. ManualSessionRecorder.StartSession()（ManualGen.Enabled == true のとき）
   e. HotkeyService.Register(HotkeyPause)
   f. ConfigChanged += OnHotkeyConfigChanged
   g. AutoStartService チェック・有効化
```

### 終了シーケンス（App.OnExit）

```
1. NotifyIconWrapper.Dispose():
   a. ConfigChanged -= OnHotkeyConfigChanged
   b. HookService.Stop()（UnhookWindowsHookEx × 2 + UnhookWinEvent）
   c. HotkeyService.Dispose()（UnregisterHotKey + DestroyHandle）
   d. TriggerOrchestrator.Dispose()（diff タイマー・キーボードアイドルタイマー廃棄）
   e. DiffDetector.Dispose()（前回サムネイルビットマップ解放）
   f. ManualSessionRecorder.StopSessionAsync().GetAwaiter().GetResult()
      → セッション未完了ステップを最終的に手順書出力
   g. NotifyIcon.Dispose() / アイコン解放
2. Mutex.ReleaseMutex() / Mutex.Dispose()
3. Log.CloseAndFlush()
```

#### `TakeAccumulatedKeys()` → `(string inputText, string keyCodes)`

```csharp
lock (_keyBufLock)
{
    string text  = _inputTextBuf.ToString();
    string codes = string.Join(", ", _keyCodeBuf);
    _inputTextBuf.Clear();
    _keyCodeBuf.Clear();
    return (text, codes);
}
```

呼び出し元: `TriggerOrchestrator.OnKeyboardActivity` のアイドルタイマーコールバック内

#### `MouseHookCallback` 詳細

```
WM_LBUTTONDOWN (0x0201):
    _lbDownTime = DateTime.UtcNow

WM_LBUTTONUP (0x0202):
    elapsedMs = (UtcNow - _lbDownTime).TotalMilliseconds
    if _lbDownTime != MinValue && elapsedMs >= DragThresholdMs:
        MouseEvent(MouseDragDrop)
    else:
        MouseEvent(MouseLeftClick)
    _lbDownTime = MinValue

WM_RBUTTONDOWN (0x0204): MouseEvent(MouseRightClick)
WM_MBUTTONDOWN (0x0207): MouseEvent(MouseMiddleClick)

WM_MOUSEWHEEL (0x020A):
    _wheelIdleTimer.Dispose() → new Timer(_ =>
        MouseEvent(MouseWheel), null, WheelIdleMs, Infinite)
```

最後に `CallNextHookEx` でフックチェーンを継続（システム動作に影響しない）。

#### `KeyboardHookCallback` 詳細

```
KBDLLHOOKSTRUCT を Marshal.PtrToStructure で取得
vk = (Keys)ks.vkCode

isDown = msg == WM_KEYDOWN(0x0100) || WM_SYSKEYDOWN(0x0104)
isUp   = msg == WM_KEYUP(0x0101)   || WM_SYSKEYUP(0x0105)

修飾キー状態更新（isDown または isUp のとき）:
    LShiftKey/RShiftKey/ShiftKey → _shiftDown
    LControlKey/RControlKey/ControlKey → _ctrlDown
    LMenu/RMenu/Menu → _altDown

if isDown:
    AccumulateKey(vk)
    KeyboardActivity イベント発火
```

#### `AccumulateKey(Keys vk)` 詳細

```
純モディファイア（Shift/Ctrl/Alt/Win 系）→ return（蓄積しない）

shift = _shiftDown, ctrl = _ctrlDown, alt = _altDown

if !ctrl && !alt:
    printable = VkToPrintable(vk, shift)

keyName = BuildKeyName(vk, shift, ctrl, alt)

lock (_keyBufLock):
    if vk == Keys.Back && _inputTextBuf.Length > 0:
        _inputTextBuf.Remove(last char)  // Backspace で削除
    else if printable != null:
        _inputTextBuf.Append(printable)
    _keyCodeBuf.Add(keyName)
```

#### `VkToPrintable(Keys vk, bool shift)` → `string?`

| 入力範囲 | 変換 |
|---|---|
| A-Z | shift ? 大文字 : 小文字 |
| Space | `" "` |
| D0-D9 + OEM 記号 | Dictionary で Shift 有無マップ参照 |
| NumPad0-9 | `"0"`-`"9"` |
| それ以外 | null |

Ctrl または Alt が押下中の場合は呼ばれない（Ctrl+C 等は印刷文字として蓄積しない）。

#### `BuildKeyName(Keys vk, bool shift, bool ctrl, bool alt)` → `string`

```
vkStr = switch(vk):
    Return → "Enter", Back → "Backspace", Delete → "Delete",
    Tab → "Tab", Escape → "Escape",
    Left/Right/Up/Down → "Left"/"Right"/"Up"/"Down"
    Home/End/Prior/Next → "Home"/"End"/"PageUp"/"PageDown"
    Space → "Space"
    F1-F24 → vk.ToString()
    A-Z → vk.ToString()
    _ → vk.ToString()

prefix = ""
if ctrl:  prefix += "Ctrl+"
if alt:   prefix += "Alt+"
if shift: prefix += "Shift+"
return prefix + vkStr
```

#### `CheckCooldown(TriggerType, double) → bool`

```csharp
lock (_lastCapture):
    if _lastCapture[type] exists:
        if (UtcNow - last).TotalSeconds < cooldownSeconds: return false
    _lastCapture[type] = UtcNow
    return true
```

クールダウン違反時は `false` を返し、呼び出し元はキャプチャをスキップする。

#### `IsExcludedApp() → bool`

```
ExcludeApps リストが空なら false

GetActiveWindowTitle() と GetActiveProcessName() を取得
ExcludeApps の各パターンに対して MatchWildcard(title, pattern) または MatchWildcard(proc, pattern)
いずれか true なら true
```

#### `MatchWildcard(string text, string pattern) → bool`

```
if pattern に '*' なし:
    text.Contains(pattern, OrdinalIgnoreCase)
else:
    Regex.Escape(pattern).Replace("\\*", ".*") でパターン生成
    Regex.IsMatch(text, "^" + regexPattern + "$", IgnoreCase)
```

#### `OnMouseEvent` → 撮影判定

```
if _paused: return
```

#### `FireCapture(TriggerType, IReadOnlyList<int>?, (string inputText, string keyCodes)?)`

```
Task.Run(async () =>
    GetCursorPos() → cursorPos
    GetActiveWindowTitle() → title
    GetActiveProcessName() → procName

    screenshots = screenIndices != null
        ? CaptureScreensByIndex(screenIndices)
        : CaptureAllScreens()

    firstMonitorPath = null
    firstMonitorBounds = default
    now = DateTime.Now

    foreach (bmp, monitorIdx, bounds) in screenshots:
        evt = new TriggerEvent(trigger, now, cursorPos, title, procName, monitorIdx)
        {
            InputText = keyboardInput?.inputText.Length > 0 ? inputText : null
            KeyCodes  = keyboardInput?.keyCodes.Length > 0  ? keyCodes  : null
        }

        if MaskPasswordFields: masking.ApplyMasking(bmp, bounds)
        if ImageOverlay:       capture.DrawImageOverlay(bmp, cursorPos, bounds, trigger)
        if BurnTimestamp:      capture.BurnTimestamp(bmp, evt.Timestamp)

        data = capture.Encode(bmp, ImageFormat, JpegQuality)
        path = await storage.SaveAsync(data, ImageFormat, evt)
        bmp.Dispose()

        if SidecarTextLog: await logger.LogEventAsync(evt, path)

        if firstMonitorPath == null:
            firstMonitorPath = path
            firstMonitorBounds = bounds

    // 手順書記録はプライマリモニターのみ
    if _manualRecorder != null && firstMonitorPath != null:
        stepEvt = new TriggerEvent(trigger, now, cursorPos, title, procName, 0)
        { InputText = ..., KeyCodes = ... }
        await _manualRecorder.RecordStepAsync(stepEvt, firstMonitorPath, firstMonitorBounds)

    _notifier.OnCaptured()
)
```

### 3.7 DiffDetector（差分検知）

**ファイル**: `Services/DiffDetector.cs`

#### フィールド

| フィールド | 型 | 説明 |
|---|---|---|
| `_captureService` | `CaptureService` | サムネイル取得 |
| `_prevThumbs` | `Dictionary<int, Bitmap>` | モニターインデックス → 前回サムネイル |
| `_lock` | `object` | `_prevThumbs` 排他ロック |

#### `DetectChangedScreens(double thresholdPercent) → List<int>`

```
for i in 0..Screen.AllScreens.Length-1:
    try:
        thumb = CaptureService.CaptureThumbnail(screens[i].Bounds)
    except Win32Exception(NativeErrorCode==6 ERROR_INVALID_HANDLE):
        // ディスプレイ利用不能（スリープ等）→ スキップして即 return
        return changed

    lock(_lock):
        if _prevThumbs[i] 存在:
            diff = CalcDiffRatio(prev, thumb)
            if diff >= thresholdPercent / 100.0:
                changed.Add(i)
                prev.Dispose()
                _prevThumbs[i] = thumb
            else:
                thumb.Dispose()  // 変化なし → 新サムネイル破棄
        else:
            _prevThumbs[i] = thumb  // 初回ベースライン設定のみ

return changed
```

#### `CalcDiffRatio(Bitmap a, Bitmap b) → double` （unsafe）

```
if a.Width != b.Width || a.Height != b.Height: return 1.0

LockBits(ReadOnly, Format32bppArgb) → pA, pB

for p in 0..Width*Height-1:
    dr = |pA[p*4+2] - pB[p*4+2]|  // R
    dg = |pA[p*4+1] - pB[p*4+1]|  // G
    db = |pA[p*4+0] - pB[p*4+0]|  // B
    if dr + dg + db > 30: diffPixels++

UnlockBits
return diffPixels / (Width * Height)
```

**閾値 30**: 3チャンネル合計で 30 超を「変化あり」とみなす。マンハッタン距離でそれぞれ約 10 の変化。

#### `RecognizeNearbyTextAsync(string imagePath, Point cursorPos, Rectangle monitorBounds)`

```
decoder = BitmapDecoder.CreateAsync(ras)
softBitmap = decoder.GetSoftwareBitmapAsync(Bgra8, Premultiplied)
```

### 3.25 ApngWriter（APNG 出力）

**ファイル**: `Services/ApngWriter.cs`

**メソッド:**

| シグネチャ | 説明 |
|---|---|
| `Task WriteAsync(string path, List<Bitmap> frames, List<double> delaysSec)` | APNG ファイルを出力 |

**APNG バイナリ構造:**

```
PNG Signature  : 89 50 4E 47 0D 0A 1A 0A
IHDR           : width(4B BE), height(4B BE), BitDepth=8, ColorType=2(RGB)
acTL           : num_frames(4B), num_plays=0(4B)  ← 0=無限ループ
[フレーム0]
  fcTL         : sequence=0, w, h, x=0, y=0, delay_num, delay_den=100, dispose=0, blend=0
  IDAT         : 圧縮済みPNGイメージデータ（PNGとして標準的に生成）
[フレーム1以降]
  fcTL         : sequence++
  fdAT         : sequence++, 圧縮済みイメージデータ
...
IEND
```

各チャンクの形式: `length(4B) | type(4B) | data | CRC32(4B)`。CRC は `type + data` のCRC32。

**遅延時間エンコード:**

`delay_num` / `delay_den = 100` で centisecond 単位。例: 3.0秒 → `delay_num = 300, delay_den = 100`。

**実装ヘルパー:**

```csharp
static void WriteChunk(Stream s, byte[] type, byte[] data)
{
    // length (big endian)
    s.Write(ToBigEndian(data.Length));
    s.Write(type);
    s.Write(data);
    // CRC32 over type + data
    s.Write(ToBigEndian(Crc32(type, data)));
}
```

```
     renderedBitmaps.Add(rendered)
6. [遅延計算]
   delays = CalcDelays(wavPaths, cfg)

7. outputDir 決定:

8. titleSlug = Regex.Replace(session.Title, "[^a-zA-Z0-9_぀-鿿]", "_").Substring(0, 40)
   baseName = $"{DateTime.Now:yyyyMMdd_HHmmss}_{titleSlug}"
9. if OutputApng:
     await ApngWriter.WriteAsync(Path.Combine(outputDir, baseName + ".png"), renderedBitmaps, delays)
10. if OutputMp4:
     await MfVideoWriter.WriteAsync(Path.Combine(outputDir, baseName + ".mp4"), ...)
11. renderedBitmaps.ForEach(b => b.Dispose())

12. _notifier?.ShowBalloon("動画生成完了", outputDir)
13. if OpenFolderOnComplete: Process.Start(new ProcessStartInfo("explorer.exe", outputDir) { UseShellExecute = true })
```

## 9. セキュリティ設計詳細

### NF-04: LLM 認証情報の保護

```
保存時（SettingsWindow.ApplySettings）:
    endpoint = PwdLlmEndpoint.Password
    cfg.ManualGen.LlmEndpoint = DpapiHelper.Protect(endpoint)
    // ProtectedData.Protect(UTF8.GetBytes(endpoint), null, CurrentUser)
    // → Base64 文字列として config.json に保存

使用時（ManualSessionRecorder.WriteSessionAsync）:
    endpoint = DpapiHelper.Unprotect(cfg.LlmEndpoint)
    // ProtectedData.Unprotect(Base64Decode(cipherText), null, CurrentUser)
    // → 平文 URL を LlmService コンストラクタに渡す（メモリ内のみ）
```

**DPAPI の特性**:
- `DataProtectionScope.CurrentUser`: 暗号化した Windows ユーザーの認証情報に紐づく
- 異なるユーザーやマシンでは復号不可
- 平文が config.json に書き込まれることはない
- ログ出力時も平文エンドポイントが出力されないよう `Unprotect` 後の値を渡さない（`LlmService` 内で `_endpoint` を `Log.Warning` に渡す際は URL として露出するが、これは Azure のエンドポイント URL であり機密度は低い）

### プライバシー機能

| 機能 | 実装 |
|---|---|
| パスワード欄マスキング | UIA `IsPasswordProperty` → 黒塗り |
| アプリ除外 | `ExcludeApps` ワイルドカードマッチ |
| ScreenDiff のマウス/キーボード直後抑制 | 1秒以内の差分は除外（意図しない自社ツールのキャプチャ防止） |

### LLM 呼び出し先の制約

- Anthropic API（api.anthropic.com）への直接接続を行わない
- `ChatCompletionsClient` の `_endpoint` は Azure AI Foundry のエンドポイント URL のみ
- クライアントライブラリ: `Azure.AI.Inference`（NuGet）
