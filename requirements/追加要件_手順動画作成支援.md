# 追加要件定義書：操作手順説明動画の自動生成

## 改訂履歴

| 版 | 日付 | 内容 |
|---|---|---|
| 1.0 | 2026-05-27 | 初版作成 |

---

## 1. 概要

### 1.1 目的

AutoScreenshot が自動撮影したスクリーンショット群を素材として、
操作の流れを視覚的に示す「操作手順説明動画」を自動生成する機能を追加する。

動画には撮影対象の強調表示・テロップ・TTS 音声を付加し、
既存の Markdown / Word 手順書と並ぶ第3の出力成果物として提供する。

### 1.2 スコープ

| 対象 | 内容 |
|---|---|
| 入力 | ManualSession（撮影済み ManualStep 列と画像ファイル） |
| 出力 | APNG ファイル、MP4 ファイル（WAV 音声付き） |
| 除外 | アニメーション GIF（実装対象外） |

---

## 2. 機能要件

### FR-V01：出力フォーマット

| ID | 要件 |
|---|---|
| FR-V01-1 | APNG 形式で動画を出力できること |
| FR-V01-2 | MP4（H.264 映像 + AAC 音声）形式で動画を出力できること |
| FR-V01-3 | APNG と MP4 はそれぞれ独立して有効/無効を設定できること |
| FR-V01-4 | アニメーション GIF は出力対象外とする |

### FR-V02：動画の構成単位

| ID | 要件 |
|---|---|
| FR-V02-1 | セッション全体を1本の動画にまとめる「セッション単位」モードをサポートすること |
| FR-V02-2 | チャプター（アクティブウィンドウ単位）ごとに別ファイルとして分割する「チャプター単位」モードをサポートすること |
| FR-V02-3 | 設定画面からいずれかを選択できること（排他選択） |

### FR-V03：フレーム表示時間

| ID | 要件 |
|---|---|
| FR-V03-1 | 全フレームを固定秒数で表示する「固定間隔」モードをサポートすること（既定 3 秒/フレーム） |
| FR-V03-2 | 撮影イベント間の実経過時間を再現する「実時間再現」モードをサポートすること |
| FR-V03-3 | 実時間再現モードでは、最小フレーム表示時間（既定 1 秒）・最大フレーム表示時間（既定 10 秒）でクリップすること |
| FR-V03-4 | 設定画面でモード選択および各秒数パラメーターを変更できること |

### FR-V04：視覚効果（強調表示）

#### FR-V04-1：クリック位置の波紋アニメーション

| ID | 要件 |
|---|---|
| FR-V04-1-1 | マウスクリックイベントのフレームで、クリック座標を中心とした波紋（リップル）を描画すること |
| FR-V04-1-2 | 波紋は中心から外側へ拡大しながらフェードアウトするアニメーションとすること |
| FR-V04-1-3 | 波紋の色はトリガー種別に応じてトリガー色設定（既存 DrawImageOverlay の配色）と統一すること |
| FR-V04-1-4 | 動画がフレーム単位（静止画ベース）の場合、波紋を複数の中間フレームに展開して表現すること |

#### FR-V04-2：操作対象コントロールの矩形枠

| ID | 要件 |
|---|---|
| FR-V04-2-1 | UIA または OCR で取得したコントロール境界矩形を半透明の矩形枠で強調表示すること |
| FR-V04-2-2 | 矩形枠の色は既存のトリガー色設定と統一すること |
| FR-V04-2-3 | 境界矩形情報がない場合は矩形枠描画をスキップすること（エラーにしない） |

### FR-V05：テロップ（字幕）

| ID | 要件 |
|---|---|
| FR-V05-1 | フレーム下部に半透明の帯を配置し、テキストをオーバーレイ描画すること |
| FR-V05-2 | 以下の情報を表示すること（表示/非表示を各項目ごとに設定可能） |
| | ・イベント種別ラベル（例：「クリック」「キー入力」「ウィンドウ切替」「差分検知」） |
| | ・入力したキー / テキスト（パスワード欄入力は「****」に置換） |
| | ・操作の説明文（LLM 改善後の ManualStep.Description。LLM 未使用時はルールベース説明文） |
| | ・タイムスタンプ（撮影日時 yyyy/MM/dd HH:mm:ss） |
| FR-V05-3 | テロップの表示位置（下部固定）・フォントサイズ（既定 16pt）・背景透明度を設定できること |
| FR-V05-4 | テロップ文字列が長い場合は折り返し表示または省略（末尾「…」）すること |

### FR-V06：音声（TTS ナレーション）

| ID | 要件 |
|---|---|
| FR-V06-1 | 各フレームの ManualStep.Description を Windows SAPI（System.Speech.Synthesis.SpeechSynthesizer）で読み上げた WAV 音声を生成すること |
| FR-V06-2 | 生成した WAV を MP4 の音声トラック（AAC）に結合すること |
| FR-V06-3 | APNG には音声を含めない（APNG は映像のみ） |
| FR-V06-4 | TTS ナレーションは設定で有効/無効を切り替えられること（既定：有効） |
| FR-V06-5 | TTS 読み上げの音声（ボイス）・速度（Rate -10〜10）・音量（0〜100）を設定できること |
| FR-V06-6 | フレームの表示時間が読み上げ時間より短い場合、フレーム表示時間を読み上げ完了まで自動延長すること（固定間隔モード時） |
| FR-V06-7 | パスワード欄のステップは「パスワードを入力しました」など固定テキストを読み上げること |

### FR-V07：生成タイミング

| ID | 要件 |
|---|---|
| FR-V07-1 | 手順書生成（Markdown / Word）と同時に動画も自動生成するモードをサポートすること |
| FR-V07-2 | 手順書生成とは独立して「動画を生成」ボタンから手動で生成するモードをサポートすること |
| FR-V07-3 | 設定画面でいずれかのモードを選択できること（既定：手動生成） |
| FR-V07-4 | 動画生成中はトレイアイコンのバルーン通知またはプログレス表示で進捗を示すこと |
| FR-V07-5 | 動画生成が完了したら出力フォルダをエクスプローラーで開く（設定で無効化可） |

### FR-V08：出力フォルダ・ファイル名

| ID | 要件 |
|---|---|
| FR-V08-1 | 動画専用の出力フォルダを設定画面で指定できること |
| FR-V08-2 | 既定の出力フォルダは `%USERPROFILE%\Videos\AutoScreenshot\` とすること |
| FR-V08-3 | ファイル名はセッション単位の場合 `{SessionTitle}_{yyyyMMdd_HHmmss}.{ext}`、チャプター単位の場合 `{SessionTitle}_{ChapterIndex:D2}_{WindowTitle}_{yyyyMMdd_HHmmss}.{ext}` とすること |

### FR-V09：出力解像度・品質

| ID | 要件 |
|---|---|
| FR-V09-1 | 出力解像度を設定画面で選択できること（選択肢：原寸 / 1920×1080 / 1280×720） |
| FR-V09-2 | 原寸を選択した場合は撮影時の画面解像度をそのまま使用すること（マルチモニター時はアクティブ画面） |
| FR-V09-3 | ダウンスケール時はアスペクト比を保持し、不足部分は黒帯でレターボックス埋めとすること |
| FR-V09-4 | MP4 の映像ビットレートを設定できること（既定：4 Mbps） |

## 3. 非機能要件

| ID | 分類 | 要件 |
|---|---|---|
| NF-V01 | 性能 | 100 ステップのセッション（HD 解像度）の動画生成が 60 秒以内に完了すること（TTS 処理時間を除く） |
| NF-V02 | 性能 | 動画生成はバックグラウンドスレッドで行い、メイン UI をブロックしないこと |
| NF-V03 | 信頼性 | 動画生成中に例外が発生した場合はエラーログを記録し、生成失敗をユーザーに通知して継続動作を維持すること |
| NF-V04 | 互換性 | Windows 10 (1809) 以降で動作すること。MediaFoundation は同バージョン以降で H.264/AAC エンコーダーが標準搭載されていることを前提とする |
| NF-V05 | セキュリティ | パスワードフィールドの画像はマスキング済み状態で動画に使用すること（MaskingService の出力を入力とする） |
| NF-V06 | 配布 | 追加の外部バイナリ（ffmpeg.exe 等）を同梱しないこと。MediaFoundation および Windows SAPI はOSに内包された API のみ使用する |

## 4. 実装方針

### 4.1 技術スタック

| 機能 | 使用技術 | 備考 |
|---|---|---|
| APNG 生成 | 純マネージド実装（PNG チャンク手書き、または NuGet `Hjson` 等の軽量ライブラリ） | 外部バイナリ不使用 |
| MP4 エンコード（映像） | Windows MediaFoundation（`MFStartup` / `IMFSinkWriter` 経由） | P/Invoke または SharpDX.MediaFoundation |
| MP4 エンコード（音声） | MediaFoundation AAC エンコーダー（`MFTransform`） | TTS で生成した PCM WAV を AAC に変換して MP4 に多重化 |
| TTS 音声生成 | `System.Speech.Synthesis.SpeechSynthesizer`（Windows SAPI） | .NET 8 では NuGet `System.Speech` が必要 |
| フレーム描画（強調表示 / テロップ） | `System.Drawing.Graphics`（GDI+）または WPF `RenderTargetBitmap` | CaptureService の既存描画処理を拡張 |

### 4.2 新規クラス設計（概要）

| クラス名 | 名前空間 | 責務 |
|---|---|---|
| `VideoGenerator` | `AutoScreenshot.Services` | 動画生成の統括。ManualSession を受け取り APNG / MP4 を出力 |
| `FrameRenderer` | `AutoScreenshot.Services` | 1フレーム分の Bitmap に波紋・矩形枠・テロップを合成する |
| `ApngWriter` | `AutoScreenshot.Services` | APNG バイナリを逐次書き込む |
| `MfVideoWriter` | `AutoScreenshot.Services` | MediaFoundation SinkWriter で MP4 ファイルを生成する |
| `TtsService` | `AutoScreenshot.Services` | SpeechSynthesizer で各ステップの WAV バイト列を生成する |
| `VideoGenConfig` | `AutoScreenshot.Models` | 動画生成に関する設定値を保持する |

### 4.3 処理フロー概要

```
VideoGenerator.GenerateAsync(ManualSession session)
├── TtsService.SynthesizeAllAsync()        // 全ステップの WAV を並列生成
├── foreach step in session.Steps
│   ├── FrameRenderer.Render(step, wav)    // 強調・テロップ合成 Bitmap を生成
│   ├── ApngWriter.AddFrame(bitmap, duration)
│   └── MfVideoWriter.AddFrame(bitmap, wav, duration)
├── ApngWriter.Finalize()                  // APNG ファイル確定
└── MfVideoWriter.Finalize()               // MP4 ファイル確定（映像+音声多重化）
```

## 5. 設定項目（VideoGenConfig）

設定ウィンドウに「動画生成」タブを新設し、以下の項目を追加する。

| カテゴリ | 設定項目 | 型 | 既定値 |
|---|---|---|---|
| 出力形式 | APNG を出力する | bool | true |
| 出力形式 | MP4 を出力する | bool | true |
| 出力フォルダ | 動画出力フォルダ | string | `%USERPROFILE%\Videos\AutoScreenshot` |
| 構成単位 | 動画単位 | enum (Session / Chapter) | Session |
| フレーム時間 | フレーム表示モード | enum (Fixed / RealTime) | Fixed |
| フレーム時間 | 固定表示秒数（秒） | double | 3.0 |
| フレーム時間 | 最小表示秒数（秒） | double | 1.0 |
| フレーム時間 | 最大表示秒数（秒） | double | 10.0 |
| 解像度 | 出力解像度 | enum (Original / FHD / HD) | Original |
| 品質 | MP4 映像ビットレート（Mbps） | int | 4 |
| 強調表示 | 波紋アニメーションを描画する | bool | true |
| 強調表示 | 矩形枠を描画する | bool | true |
| テロップ | テロップを表示する | bool | true |
| テロップ | イベント種別ラベルを表示する | bool | true |
| テロップ | 入力テキストを表示する | bool | true |
| テロップ | 説明文を表示する | bool | true |
| テロップ | タイムスタンプを表示する | bool | true |
| テロップ | フォントサイズ（pt） | int | 16 |
| テロップ | 背景透明度（0〜255） | byte | 160 |
| TTS | TTS ナレーションを有効にする | bool | true |
| TTS | 音声（ボイス名） | string | （OS既定） |
| TTS | 読み上げ速度（-10〜10） | int | 0 |
| TTS | 音量（0〜100） | int | 100 |
| 生成タイミング | 手順書生成と同時に動画を自動生成する | bool | false |
| 完了時 | 生成完了後にエクスプローラーで開く | bool | true |

## 6. 制約・前提条件

| # | 内容 |
|---|---|
| 1 | MediaFoundation の H.264 エンコーダー（`MFT_CATEGORY_VIDEO_ENCODER`）は Windows 10 1809 以降に標準搭載。対応 OS 要件（Windows 10 1809+）は変更しない |
| 2 | Windows SAPI（`System.Speech`）は .NET 8 では `System.Speech` NuGet パッケージ（`net8.0-windows` TFM）が必要 |
| 3 | APNG は FireFox / Chrome 等のモダンブラウザでネイティブ再生可能。Word / Edge での表示は動作確認対象外 |
| 4 | LLM（Azure AI Foundry）との連携方針は既存の非機能要件（NF-04）に準拠。本機能でも API キーの平文保存は禁止 |
| 5 | 動画生成に使用する画像は MaskingService 適用済みの保存済みスクリーンショットから読み込む。リアルタイムキャプチャは行わない |
| 6 | WebP 形式で保存されたスクリーンショットは `SixLabors.ImageSharp` で PNG/JPEG にデコードしてから GDI+ に渡す |
| 7 | ffmpeg.exe・VLC 等の外部バイナリは同梱しない。OS 組み込み API（MediaFoundation / SAPI）のみで完結させる |
