# AutoScreenshot v1.7.0 リリースノート

**リリース日**: 2026-05-30

---

## 概要

画像品質の向上（ロスレス保存）、保存先パスの柔軟なテンプレート設定、
フォールバックフォルダによる可用性向上、エクスポート進捗ダイアログを追加した
メジャーアップデートです。

---

## 変更点

### Added（追加）

#### ロスレス画像保存（FR-H1・FR-H2）

- **before 画像**（操作前）を PNG 固定から設定フォーマット（PNG/JPEG/WebP）に揃え、
  ロスレス品質で保存するよう変更
  - JPEG: quality = 100（無圧縮）
  - WebP: lossless モード
- **after 画像**（操作後）も同様にロスレスで保存

#### 保存先フォルダのテンプレート設定（FR-H3）

保存先を「ベースフォルダ ＋ 日付テンプレートフォルダ ＋ ファイル」の3階層で構成できます。

**使用可能なプレースホルダー:**

| プレースホルダー | 例（2026-05-30 14:30） |
|---|---|
| `{date}` | `20260530` |
| `{datetime}` | `20260530143022` |
| `{date_time}` | `20260530_143022` |
| `{date-time}` | `20260530-143022` |
| `{hour}` | `1400` |
| `{title}` | `操作手順書_2026_05_30` |
| `{title_short}` | タイトル40文字まで |
| `{id}` | セッションID先頭8文字 |
| `{username}` | ユーザー名 |
| `{computername}` | PC名 |

**設定例:**
- プロジェクトフォルダ: `{date_time}_{title_short}.ascproj`
- 手順書フォルダ: `{date_time}`
- 動画フォルダ: `yyyyMMdd-HH00-顧客定例会`

#### 「パス設定」タブ（FR-H4）

設定ウィンドウの「一般」タブの直後にパス設定専用タブを追加。
画像・手順書・動画のベースフォルダとテンプレートを一画面で設定できます。
[ℹ] アイコンにマウスを合わせるとプレースホルダー一覧と使用例を確認できます。

#### 第2ベースフォルダ（フォールバック）（FR-H5）

ドライブ障害・容量不足など書き込み不能時に自動で第2フォルダへ切り替えます。

- アプリ起動時: ドライブ存在確認・書き込みテストを実施
- 撮影中: 書き込み失敗を検知したら即座に第2フォルダへ切り替え
- 切り替え時はバルーン通知でユーザーに通知

#### エクスポート進捗ダイアログ（FR-H6）

エクスポート処理中にノンモーダルの進捗ウィンドウを表示します。

- プログレスバーと処理カウンター（ステップ数/フレーム数）を表示
- キャンセルボタンで処理を中断可能
- エクスポート中はトレイアイコンのツールヒントにも進捗を表示

### Changed（変更）

- バージョン: 1.6.1 → 1.7.0
- 設定ウィンドウ: パス関連設定を「パス設定」タブに集約

## アップグレード時の注意

- 旧 `config.json` の `SaveFolder`（保存フォルダ）は `ImageBaseFolder` に自動移行されます
- 旧 `FolderNaming`（フォルダ命名規則）は廃止され、`ImageFolderTemplate` に置き換わります。
  既定値 `yyyyMMdd` が使用されます
- `.ascproj` フォルダはそのまま使用可能です
