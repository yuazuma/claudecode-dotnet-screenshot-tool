# AutoScreenshot v1.4.0 リリースノート

**リリース日**: 2026-05-28

---

## 概要

v1.4.0 は UI/UX の改善と視覚フィードバック強化に特化したリリースです。
タスクトレイアイコンが処理状態に応じて 5 色に変化し、プロジェクトビューウィンドウは
ステップ一覧と詳細を常時並列表示する新レイアウトに刷新されました。

---

## 新機能・変更点

### タスクトレイアイコン 5 状態表示

処理状態をアイコン色でリアルタイムにフィードバックします。

| 状態 | 色 | タイミング |
|---|---|---|
| Recording | 青 `#0078D7` | 通常記録中 |
| Paused | グレー `#808080` | 一時停止中・ディスク不足自動停止 |
| Captured | 緑 `#107C10` | 撮影成功後 200ms フラッシュ |
| Processing | オレンジ `#CA5010` | エクスポート処理中（複数並走対応） |
| Error | 赤 `#C50F1F` | エクスポートエラー後 5 秒間表示 |

### トレイメニュー再構成

プロジェクト機能の有効/無効に応じてメニュー構成を切り替えます。
- プロジェクト機能有効時: プロジェクト名・エクスポートサブメニューを表示
- プロジェクト機能無効時: セッション管理・手順書生成・動画生成を直接表示

### 設定ウィンドウ タブ順序変更

よく使うタブを前方に整理しました。

```
[一般] [撮影トリガー] [保存] [プライバシー]    ← キャプチャ設定
[手順書生成] [LLM連携] [動画生成] [プロジェクト]  ← 出力・生成
[通知] [メタデータ]                           ← 補助・詳細
```

### プロジェクトビューウィンドウ UI リファクタリング

- **縦型ステップ一覧**: ステップ番号・サムネイル・説明文を常時表示。ドラッグ並び替えハンドル（☰）付き
- **常時分割レイアウト**: ステップ一覧と詳細（画像・アノテーション・説明文）を同時表示。タブ切り替え不要
- **エクスポートドロップダウン**: ウィンドウ上部の「▼ エクスポート」ボタンに全エクスポート機能を集約
- **アノテーション Expander**: 既存アノテーションがある場合のみ自動展開。画面を広く使用可能
- **進捗バー**: エクスポート処理中にステータスバーで ProgressBar を表示
- **削除済みを表示**: チェックボックスで削除済みステップの表示/非表示を切り替え

## インストール

1. `AutoScreenshot-v1.4.0-win-x64.zip` を展開して任意のフォルダに配置
2. `AutoScreenshot.exe` を実行（管理者権限・.NET ランタイム不要）
3. v1.3.0 からのアップグレード: 既存の `config.json` と `.ascproj` フォルダはそのまま使用可能
