AutoScreenshot v1.0.0
=====================

タスクトレイ常駐型 自動スクリーンショット撮影・操作手順書作成ツール

【動作環境】
- Windows 10 (1809) 以降 / Windows 11
- x64 アーキテクチャ
- .NET ランタイム不要（自己完結型バイナリ）

【インストール方法】
1. ZIP を任意のフォルダに展開する
2. AutoScreenshot.exe を実行する
3. タスクトレイにアイコンが表示されれば起動完了

【アンインストール方法】
1. タスクトレイアイコンを右クリック → 終了
2. 展開したフォルダをすべて削除する
3. 設定ファイルを削除する場合: %APPDATA%\AutoScreenshot\ を削除する
4. 自動起動エントリを削除する場合:
   レジストリ HKCU\SOFTWARE\Microsoft\Windows\CurrentVersion\Run から
   AutoScreenshot エントリを削除する

【初回起動時の注意】
- Windows セキュリティの「SmartScreen」警告が表示される場合は、
  「詳細情報」→「実行」をクリックしてください。
- 管理者権限は不要ですが、グローバルキーボード/マウスフックの設置に
  ユーザー権限が必要です（通常は問題ありません）。

【主な機能】
- マウスクリック・キーボード入力・ウィンドウ切替・画面差分を自動検知して撮影
- 撮影した画像から操作手順書（Markdown / Word）を自動生成
- Azure AI Foundry (Claude) を使った操作説明文の自然言語改善
- パスワード欄の自動マスキング
- タスクトレイから一時停止・設定変更
