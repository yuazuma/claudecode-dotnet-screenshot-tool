# AutoScreenshot v1.3.0 リリースノート

**リリース日**: 2026-05-27

---

## 概要

v1.3.0 では、スクリーンショットへの視覚的注釈付け・プロジェクト管理強化・結合/分割操作・HTML エクスポート・インクリメンタル LLM 処理の 5 機能を追加しました。

---

## 新機能

### FR-C: ステップアノテーション

プロジェクトビューウィンドウの右ペインにアノテーションパネルを追加しました。

- **ツール**: 番号バッジ / 矢印 / 矩形 / 吹き出しの 4 種類
- **色**: 赤 / 青 / 黄 / 緑
- アノテーションは `project.json` の `annotations` フィールドに座標付きで保存
- エクスポート（Markdown / Word / HTML / 動画）では自動的に焼き込み済み画像を使用

### FR-D: プロジェクト管理強化

- **全文検索**: タイトル・説明文・タグで絞り込み
- **タグフィルタ**: WrapPanel のトグルボタンで複数タグ絞り込み
- **ステップ並び替え**: サムネイル一覧をドラッグ&ドロップで並び替え（StepNumber 自動更新）
- **手動ステップ挿入**: 「＋ステップを追加」ボタンで任意の画像と説明文を追加

### FR-E: プロジェクト結合・分割

- **結合**: 複数プロジェクトを選択して「結合...」ボタン → タイトル入力 → 時系列順に統合した新プロジェクトを生成（元プロジェクトは変更されません）
- **分割**: ステップを選択して「ここで分割」ボタン → 2 つのタイトル入力 → 指定ステップを境に前後 2 プロジェクトを生成（元プロジェクトは変更されません）

### FR-A: HTML エクスポート

- 「手順書 (HTML)」ボタンで単一 HTML ファイルを生成
- 画像は Base64 埋め込みのため、受け渡しがファイル 1 枚で完結
- 記録停止時の自動 HTML エクスポートを設定ウィンドウで有効化可能

### FR-B: インクリメンタル LLM 処理

- ステップ追記後に自動的に LLM 説明文改善をバックグラウンドキューで処理
- 設定ウィンドウの「ステップ追記後に自動 LLM 改善」チェックボックスで有効化
- LLM 失敗時はルールベース説明文でフォールバック（既存動作維持）

## インストール

1. `AutoScreenshot-v1.3.0-win-x64.zip` を展開して任意のフォルダに配置
2. `AutoScreenshot.exe` を実行（管理者権限・.NET ランタイム不要）
3. v1.2.0 からのアップグレード: 既存の `config.json` と `.ascproj` フォルダはそのまま使用可能
